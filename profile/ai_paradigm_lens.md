# User Profile — AI & Data Engineering Lens

## Role
Senior Data/AI Engineer. Primary interests: building production AI systems, data pipelines,
and staying ahead of paradigm shifts in LLM tooling.

## High-value topics (score 4–5)
- Large Language Models: new architectures, training techniques, inference optimization
- Agentic AI: multi-agent frameworks, tool use, MCP, autonomous pipelines
- Data Engineering: modern lakehouse patterns, streaming, dbt, Spark, Databricks
- MLOps / LLMOps: evaluation, observability, deployment, fine-tuning at scale
- Azure AI: Foundry, Azure OpenAI, Semantic Kernel, Agent Framework
- Open-source releases: major model drops, framework 1.0s, breaking changes

## Medium-value topics (score 3)
- Cloud infrastructure changes that affect AI workloads (pricing, quotas, new services)
- Industry news: major acquisitions, pivots, or shutdowns in AI tooling space
- Research papers with clear engineering implications (not just benchmark leaderboards)

## Low-value / noise (score 1–2)
- Pure frontend/mobile development
- Marketing content without technical depth
- "AI is transforming [industry]" think pieces without concrete details
- Benchmark comparisons without methodology
- Announcements of future products without availability

## Output language
English throughout.
